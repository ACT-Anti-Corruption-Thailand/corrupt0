# Data

Folder นี้เอาไว้เก็บข้อมูลดิบ + Script ที่ใช้ transform ข้อมูลเหล่านั้น

## Fetch

ดึงข้อมูลด้วยคำสั่ง

```bash
pnpm fetch
```

## Process

Process ข้อมูลด้วยคำสั่ง

```bash
pnpm process
```
